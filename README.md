###### How to run application

```
//Install node from https://nodejs.org/en/download/

### set up application
npm install -g bower
bower i
npm i

### Run application
PORT=8000 nodemon index.js

```


###### Create your .env file in the main folder for development

```
DATABASE_URL=
DATABASE_NAME=
DATABASE_USER=
DATABASE_PW=

S3_BUCKET=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_REGION=
```
